# Project Overview

This projects is a personal portfolio website showcasing projects, skills, and contact information.

## Core Domain

- **Personal Branding**: Presents the developer's identity and skills.
- **Project Showcase**: Display case studies or links to past work.
- **Contact Mechanism**: Simple way for visitors to reach out.

## Key Directories

- `src/`: Source code for the frontend application.
- `public/`: Static assets.
- `.context/`: AI-focused documentation context.
