# Style Guide

## General Patterns

- **React Components**: Functional components with hooks.
- **File Structure**: `src/components` for reusable UI, `src/pages` for routes.

## CSS / Styling

- **Primary**: Tailwind CSS utility classes.
- **Secondary**: Ant Design components for complex interactions.
- **Collision Avoidance**: Use specific class names or modules if custom CSS is needed.

## Naming Conventions

- **Components**: PascalCase (e.g., `MyComponent.js`).
- **Utilities**: camelCase (e.g., `formatDate.js`).
- **Constants**: UPPER_SNAKE_CASE (e.g., `API_URL`).
